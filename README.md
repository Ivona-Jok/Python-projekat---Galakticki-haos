# Galaktički haos

Projekat je razvijen kao timski zadatak za predmet na fakultetu koristeci programski jezik Python.

## Opis

Igrač upravlja brodom u svemiru i bori se protiv neprijatelja kroz 5 nivoa. Svaki nivo donosi veću težinu, brže pucanje i više projektila istovremeno.

## Instalacija

1. Kloniraj repozitorijum:
   https://github.com/Ivona-Jok/Python-projekat---Galakticki-haos.git
2. Napravi virtuelno okruženje:
   py -3.12 -m venv .venv
  .venv\Scripts\activate
3. Instaliraj pakete:
   pip install -r requirements.txt
4. Pokreni igricu:
   python main.py

## Kontrole

| Tipka | Akcija |
|-------|--------|
| W A S D | Kretanje broda |
| Miš | Ciljanje mišom |
| Desni klik (držanje) | Pucanje |
| ESC | Izlaz iz igrice |
| + / - | Promjena nivoa (testiranje) |

## Tehnologije

- Python 3.12
- Pygame 2.6.1
- SQLite3
